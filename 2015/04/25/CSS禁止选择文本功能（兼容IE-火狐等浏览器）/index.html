有时候，我们为了用户体验，需要禁用选择文本功能。
这需要用到一个CSS属性：user-select，user-select的文档点这里

user-select 有两个值：
none 用户不能选择文本
text 用户可以选择文本

需要注意的是：user-select并不是一个W3C的CSS标准属性，浏览器支持的不完整，需要对每种浏览器进行调整

IE6-9还没发现相关的CSS属性，只能通过JS来实现

这样我们就可以结合CSS和JS的方法来禁用浏览器的文本选择功能了
